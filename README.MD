# SDL3 Sandbox

A minimal SDL3 project template for C on Windows (x64) using MinGW.  
This project demonstrates a basic SDL3 setup with a window, renderer, event loop, and simple rectangle drawing.

---

## Features

- Open a resizable SDL3 window
- Renderer-based 2D drawing
- Event handling (close window, ESC key)
- FPS-friendly main loop (~60 FPS)
- Minimal and easily expandable template

---

## Requirements

- **Windows x64**
- **MinGW-w64** (x86_64)
- **SDL3 Development Package**  
  Download from [SDL official site](https://www.libsdl.org/SDL-3.2.28.zip)
- **SDL3.dll** must be in the same folder as `app.exe` at runtime
